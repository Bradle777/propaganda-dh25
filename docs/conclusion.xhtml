Forming Conclusions

The core of this project is to create a tool with which one can generate novel conclusions.

While we have presented statistical overlap between attributes in the Analysis section, those patterns serve more as a standard to identify consistent patterns across the 43-year corpus.

Below, we describe three major lenses through which to interpret rhetoric patterns and form conclusions.

Political Relativist View

Follows the logic that political propaganda is mostly in response to the contemporary political climate.
